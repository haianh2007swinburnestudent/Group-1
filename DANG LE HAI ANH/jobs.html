Reference: A1B2C – Junior Web Developer

Salary range: $55,000 – $70,000

Reports to: Lead Developer

Brief description

The Junior Web Developer assists in designing, developing, and maintaining web applications while learning modern front-end technologies under the guidance of senior developers.

Key responsibilities

  1. Develop responsive web pages using HTML, CSS, and JavaScript.
  2. Fix bugs and conduct unit testing to ensure code quality.
  3. Collaborate with designers to enhance user experience.
  4. Participate in code reviews and maintain project documentation.

Qualifications, Skills, and Attributes

Essential

  • Bachelor’s degree in Computer Science or equivalent.
  • Strong understanding of HTML, CSS, and JavaScript.
  • Excellent teamwork and problem-solving skills.

Preferable

  • Experience with React.js or Vue.js frameworks.
  • Basic knowledge of version control systems (Git).

Reference: Z9X8Y – Cybersecurity Analyst

Salary range: $75,000 – $95,000

Reports to: Security Manager

Brief description

The Cybersecurity Analyst is responsible for monitoring company networks, investigating suspicious activities, and implementing preventive security measures.

Key responsibilities

  1. Monitor and analyze security events from multiple sources.
  2. Identify, respond to, and mitigate cyber threats.
  3. Conduct regular vulnerability testing and audits.
  4. Support staff by promoting cybersecurity awareness.

Application Information

Applications for all positions close on 30/11/2025.

To apply, please visit the Job Application Page.

Qualifications, Skills, and Attributes

Essential

  • Bachelor’s degree in Cybersecurity, IT, or related field.
  • Knowledge of network protocols, firewalls, and intrusion detection systems.
  • Experience with SIEM tools such as Splunk or Azure Sentinel.

Preferable

  • Certifications: CompTIA Security+, CEH, or CISSP.
  • Understanding of cloud security principles (AWS, Azure).
